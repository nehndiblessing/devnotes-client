# DevNotes

DevNotes is a clean and responsive note-taking web application built using:

- HTML
- CSS
- JavaScript

It supports:

- Create notes
- Edit notes
- Delete notes
- Pin/unpin notes
- Search notes
- Dark/light theme
- Live preview
- Local storage persistence
- Notes statistics

---

# Features

## Notes Management

- Add new notes
- Edit existing notes
- Delete notes
- Pin important notes

## Search

Search notes instantly by:

- Title
- Content

## Theme Toggle

Switch between:

- Light Mode
- Dark Mode

Theme preference is saved automatically.

## Live Preview

Preview note content while typing.

## Statistics

Displays:

- Total notes
- Total characters

## Local Storage

All notes are saved in browser localStorage.
